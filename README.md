# 🍜 Void Restaurant Website

A fully animated, multi-page restaurant website
built with pure HTML, CSS, and JavaScript.
No frameworks. No libraries.

## 🌐 Live Demo
[View Live Site]([https://YOURUSERNAME.github.io/void-restaurant](https://nishanrokaya535-png.github.io/void-restaurant-))

## ✨ Features
- Custom animated cursor
- Smooth page loader
- Parallax hero background
- 3D card tilt on hover
- Scroll reveal animations
- Marquee text strip
- Gallery with lightbox
- Reservation form (Formspree)
- Mobile responsive
- Red & Gold Asian aesthetic

## 📁 Pages
- `index.html` — Homepage
- `about.html` — Our Story & Team
- `services.html` — Full Menu
- `contact.html` — Reservations & Contact
- `styles.css` — All styling
- `script.js` — All animations & interactions

## 🛠️ Built With
- HTML5
- CSS3 (animations, variables, grid, flexbox)
- Vanilla JavaScript
- Google Fonts
- Formspree (forms)
